# ReadyCpp

**ReadyCpp** est une bibliothèque de fonctions C++ pour la Programmation Orientée Objet en C++.  
C'est une bibliothèque développée en C++ et maintenu par 
[Gerard KESSE](http://5.48.230.127:8855/presentation/ "Accédez à mon site web (ReadyDev)").

# Technologies

Le développement du package **ReadyCpp** a nécessité les technologies suivantes :

* Langages : **C++**
* IDEs : **Visual Studio**
* Outils de Construction: **NMake**
* Editeurs de Texte: **Notepad++**
* Concepts : **Classe, Héritage, Polymorphisme, Iterateur, Foncteur, STL, Template**
* Cible : **Windows, UNIX**
